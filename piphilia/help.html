This is a memory game! You progress through the game by clicking the next correct element in the bottom row. Clicking the correct element removes that row. Using Pi sequence as an example, first you'd click 3, then 1, then 4. If an incorrect element is clicked, that block will be changed to red. The Pi sequence is not infinite but if you're able to get through the 3142 digits, kudos! You win.
